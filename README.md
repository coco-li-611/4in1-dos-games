## 📖 项目概述

这是一个用 **MASM/TASM 汇编语言** 编写的纯DOS游戏合集，包含四个经典小游戏。所有游戏都直接运行在x86实模式架构上，展示了底层系统编程的强大能力。

### 游戏列表

| # | 游戏名称 | 英文名 | 核心机制 | 技术亮点 |
|---|---------|--------|---------|---------|
| 1 | 🃏 记忆翻牌 | Memory Match | 翻牌配对记忆 | Fisher-Yates洗牌算法 |
| 2 | 🐍 贪吃蛇 | Snake | 经典贪吃蛇 | 中断Hook + 直接显存写入 |
| 3 | ⌨️ 打字测试 | Typing Speed Test | 打字速度测试 | 精确计时系统 |
| 4 | 🏰 迷宫探索 | Maze | 迷宫逃生 | 碰撞检测算法 |

## 🎯 游戏详情

### 🃏 Game 1: 记忆翻牌 (Memory Match)

```
┌─────────────────────────────────────────────┐
│                                             │
│    ╔═══╗  ╔═══╗  ╔═══╗  ╔═══╗            │
│    ║ ? ║  ║ ? ║  ║ ? ║  ║ ? ║   Row 10   │
│    ╚═══╝  ╚═══╝  ╚═══╝  ╚═══╝            │
│                                             │
│    ╔═══╗  ╔═══╗  ╔═══╗  ╔═══╗            │
│    ║ ? ║  ║ 3 ║  ║ ? ║  ║ ? ║   Row 12   │
│    ╚═══╝  ╚═══╝  ╚═══╝  ╚═══╝            │
│                                             │
│    Select card (1-8):                       │
│                                             │
└─────────────────────────────────────────────┘
```

**游戏规则**: 翻开两张卡片，如果数字相同则配对成功，否则翻回。找到所有配对即获胜。

**技术实现**:
- `RC_SHUFFLE_CARDS`: Fisher-Yates 洗牌算法
- `RC_DRAW_GRID`: 3态卡片系统（隐藏/翻开/配对）
- `RC_DELAY`: 精确延时（INT 1Ah 时钟中断）

```

│ Score=0000H                                        direction: R    │
│                                                                    │

│                                                                    │
│                       ████████                                     │

│                                                                    │
│                                                                    │
│                                                                    │
│ ════════════════════════════════════════════════════════════════  │
└────────────────────────────────────────────────────────────────────┘
                    Press SPACE to Start Game!
```

**游戏规则**: 使用方向键控制蛇的移动，吃到食物增长蛇身，撞墙或撞自己则游戏结束。

**技术实现**:
- `SN_new_int9`: 键盘中断Hook (INT 09h)
- 直接写屏 B800h: 显存直接操作，无闪烁
- 链表数据结构: 动态管理蛇身节点
- 方向防反转: 防止180度调头

### ⌨️ Game 3: 打字速度测试 (Typing Speed Test)

```
┌─────────────────────────────────────────────┐
│                                             │
│  ===== Typing Speed Test =====              │
│                                             │
│  Type the characters as fast as you can!    │
│  You have 10 seconds.                       │
│                                             │
│            M                                │
│                                             │
│  Correct: 23                                │
│  Total typed: 25                            │
│  Accuracy: 92%                              │
│  Speed: 25 chars/sec                        │
│                                             │
└─────────────────────────────────────────────┘
```

**游戏规则**: 在10秒内尽可能多地输入屏幕上显示的字符。统计正确率与打字速度。

**技术实现**:
- `INT 1Ah`: BIOS时钟服务，精确计时
- 字符池算法: 36字符随机选择
- 实时统计: 正确数/总数计算
- `TY_PRINT_NUM`: 递归数字打印

### 🏰 Game 4: 迷宫游戏 (Maze)

```
┌─────────────────────────────────────────────┐
│ ####################                        │
│ #S        #     #                           │
│ #####  #  #  #  #                          │
│ #    #  #  #  #  #                          │
│ #  ####  #  ####  #                          │
│ #     #  #     #  #                          │
│ #####  #####  ####  #                          │
│ #     #     #     #  #                          │
│ #  #########  #####  #                          │
│ #E               #                          │
│ ####################                          │
│                                             │
│  W=Up S=Down A=Left D=Right                 │
│                                             │
└─────────────────────────────────────────────┘
     S = 起点 (@ = 玩家位置)   E = 终点
```

**游戏规则**: 使用WASD键控制玩家(@)从起点(S)移动到终点(E)。

**技术实现**:
- `check_wall`: 碰撞检测，防止穿墙
- 2D数组地图: 10x10字符迷宫
- 状态刷新: 实时显示玩家位置

## 🔧 技术亮点

### 系统级编程技术

```asm
; ═══════════════════════════════════════════════════════════
; BIOS 中断调用
; ═══════════════════════════════════════════════════════════
INT 10h          ; 显示服务 - 屏幕清空、光标定位、字符输出
INT 16h          ; 键盘服务 - 按键读取（主菜单/翻牌/迷宫）
INT 09h          ; 硬件中断 - 贪吃蛇方向控制（Hook）
INT 1Ah          ; 时钟服务 - 计时、随机数种子
INT 21h          ; DOS服务  - 字符串显示、程序退出

; ═══════════════════════════════════════════════════════════
; 直接显存操作
; ═══════════════════════════════════════════════════════════
MOV AX, 0B800h   ; 设置显存段地址
MOV ES, AX       ; 贪吃蛇高速渲染
MOV ES:[DI], DX  ; 直接写入像素

; ═══════════════════════════════════════════════════════════
; 随机数生成
; ═══════════════════════════════════════════════════════════
MOV AX, [SEED]   ; 获取种子
MUL MULTIPLIER   ; 线性同余
ADD AX, 3        ; 偏移
DIV CX           ; 取余得到范围[0, CX)
```

### 数据结构设计

| 模块 | 数据结构 | 说明 |
|------|----------|------|
| 贪吃蛇 | 双向链表 | 节点包含: 前指针、后指针、显存位置、颜色 |
| 记忆翻牌 | 状态机 | 3态系统: HIDDEN(0) → REVEALED(1) → MATCHED(2) |
| 迷宫 | 2D数组 | 10x10 char Map[#墙 .空 S起点 E终点] |
| 打字测试 | 字符池 | 36字符预定义池，随机索引访问 |

### 高级算法

```
┌─────────────────────────────────────────────────────────────┐
│ Fisher-Yates 洗牌算法 (记忆翻牌)                            │
├─────────────────────────────────────────────────────────────┤
│ for i = n-1 downto 1:                                      │
│     j = random() % (i + 1)                                 │
│     swap(array[i], array[j])                               │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 键盘中断Hook流程 (贪吃蛇)                                   │
├─────────────────────────────────────────────────────────────┤
│ 1. 保存原始INT 09h向量                                      │
│ 2. 安装自定义中断处理程序                                   │
│ 3. 读取端口60h获取扫描码                                   │
│ 4. 更新方向变量                                            │
│ 5. 调用原中断处理                                          │
└─────────────────────────────────────────────────────────────┘
```

## 💻 编译与运行

### 环境要求

| 组件 | 版本 | 说明 |
|------|------|------|
| MASM | 6.15+ | Microsoft Macro Assembler |
| TASM | 5.0+ | Turbo Assembler (可选) |
| DOSBox | 0.74+ | DOS模拟器 |
| 操作系统 | Windows/Linux/macOS | 运行DOSBox |

### 编译步骤

```bash
# 方式一: MASM + LINK
masm 4.asm;
link 4;
4.exe

# 方式二: TASM + TLINK
tasm 4.asm
tlink 4.obj
4.exe

# 方式三: 一键编译脚本
compile.bat
```

### 运行

```bash
# 启动DOSBox
dosbox

# 在DOSBox中
MOUNT C D:\dosbox_work
C:
4.EXE
```
